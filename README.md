# کوچ تحصیلی هوشمند (فاز اول)

بات تلگرامی که نقش یک کوچ تحصیلی برای دانش‌آموز پایه دوازدهم رشته تجربی (در
حال آماده‌سازی برای کنکور سراسری) رو ایفا می‌کنه. فاز اول شامل چت هوشمند آزاد
با AI و چک‌این شبانه با دکمه‌های اینلاینه.

## قابلیت‌های فاز اول

- چت آزاد با هوش مصنوعی (DeepSeek) با حفظ تاریخچه مکالمه
- چک‌این شبانه خودکار (ساعت مطالعه، مباحث خوانده‌شده، حال روحی)
- ذخیره‌سازی در دیتابیس SQLite

## پیش‌نیازها

- Python 3.11 یا بالاتر
- یک توکن بات تلگرام (از [@BotFather](https://t.me/BotFather))
- یک کلید API از [DeepSeek](https://platform.deepseek.com)

## نصب

### ۱. ساخت virtual environment

در پوشه پروژه (`konkur-coach`) دستور زیر را اجرا کنید:

**ویندوز (PowerShell):**

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
```

**مک / لینوکس:**

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### ۲. نصب پکیج‌ها

```bash
pip install -r requirements.txt
```

### ۳. تنظیم فایل `.env`

فایل `.env.example` را کپی کرده و به نام `.env` ذخیره کنید:

```bash
cp .env.example .env
```

سپس فایل `.env` را باز کرده و مقادیر زیر را با اطلاعات واقعی خودتان پر کنید:

| متغیر | توضیح | نمونه |
|---|---|---|
| `TELEGRAM_BOT_TOKEN` | توکنی که از @BotFather می‌گیرید | `123456:ABC-DEF...` |
| `AI_API_KEY` | کلید API حساب DeepSeek شما | `sk-...` |
| `AI_BASE_URL` | آدرس پایه API (پیش‌فرض DeepSeek) | `https://api.deepseek.com` |
| `AI_MODEL` | نام مدل | `deepseek-chat` |
| `CHECKIN_HOUR` / `CHECKIN_MINUTE` | ساعت ارسال چک‌این شبانه | `22` / `0` |
| `TIMEZONE` | منطقه زمانی چک‌این | `Asia/Tehran` |

**مهم:** فایل `.env` هرگز نباید به گیت‌هاب push بشه (در `.gitignore` قرار داره).

## اجرای بات

```bash
python main.py
```

اگر همه‌چیز درست باشه، در لاگ پیام `بات در حال اجراست...` رو می‌بینید.

## تست سریع

1. بعد از اجرای بات، در تلگرام به بات خودتون پیام `/start` بفرستید — باید پیام
   خوش‌آمدگویی دریافت کنید.
2. یک پیام آزاد مثل «چطور برای زیست‌شناسی برنامه‌ریزی کنم؟» بفرستید — باید پاسخ
   هوش مصنوعی رو دریافت کنید.
3. برای تست فلوی چک‌این بدون منتظر موندن تا ساعت مشخص‌شده، می‌تونید موقتاً
   `CHECKIN_HOUR` و `CHECKIN_MINUTE` رو در `.env` روی چند دقیقه‌ی بعد از الان
   تنظیم کنید و بات رو مجدداً اجرا کنید تا پیام چک‌این با دکمه‌ها ارسال بشه.

## ساختار پروژه

```
konkur-coach/
├── main.py                  # نقطه ورود اصلی
├── config.py                 # تنظیمات و متغیرهای محیطی
├── ai/
│   ├── client.py              # ارتباط با DeepSeek/OpenAI-compatible API
│   └── system_prompt.py       # شخصیت کوچ
├── database/
│   ├── models.py              # جدول‌های users و daily_checkins
│   └── db.py                  # توابع CRUD
├── handlers/
│   ├── chat_handler.py        # چت آزاد
│   └── checkin_handler.py     # فلوی چک‌این
└── scheduler/
    └── daily_job.py            # زمان‌بند چک‌این شبانه
```

## نکته درباره سوییچ AI Provider

کد به هیچ provider خاصی قفل نیست. کافیه در `.env` مقدار `AI_BASE_URL` و
`AI_MODEL` رو عوض کنید (مثلاً برای Anthropic یا هر سرویس دیگه‌ی سازگار با
فرمت OpenAI) — نیازی به تغییر کد نیست.

## نقشه راه (فازهای بعدی)

فاز اول عمداً محدود نگه داشته شده. موارد زیر در فازهای بعدی اضافه می‌شن:

- پنل وب برای کوچ انسانی
- گزارش‌دهی هفتگی خودکار برای والدین
- چندکاربره واقعی و احراز هویت
- مهاجرت دیتابیس به PostgreSQL
